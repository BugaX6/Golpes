# 🛡️ Sistema de Detecção de Golpes Financeiros

Este é um projeto desenvolvido em **Java** que simula um detector de phishing e golpes financeiros. O objetivo é educar usuários sobre segurança digital e boas práticas para evitar fraudes online.

## 🚀 Tecnologias Utilizadas
- **Linguagem:** Java
- **Bibliotecas:** `java.util.*`, `java.time.*`

## 🛠️ Funcionalidades
- Menu interativo com lista de golpes comuns.
- Lógica de detecção baseada em padrões de phishing.
- Formatação de datas e horários para logs de segurança.
